Spacial Anchor System for AR Layoutplanning

Developed @MAN Energy Solutions (Everllence)

Project Overview

  • Engine: Unity
  • Programming Language: C#
  • Used Hardware: Meta Quest 3
  • Team Size: 1
  • Status: finished

Descritpion

During my working student position at MAN Energy Solutions, the Meta Quest 3 was released, introducing the ability to scan and save a room mesh. At the same time, MAN was planning the setup of a new production hall, which was to be digitally arranged in advance with the help of AR. This setup, however, needed to be quite precise, as it was necessary to verify whether certain machine positions made sense (ensuring there was enough space to operate them). Conventional methods, such as manually placing visual markers, can of course lead to inaccuracies. That’s when I had the idea to anchor the AR system to the scanned corners of the room using the Meta Quest 3 mesh. The system was very well received, especially since, just a few weeks after the release of the Meta Quest 3, we already had an in-house, meaningful further development of this hardware available.

My Tasks

  • Planning
  • Implementation of the System
  • 3D Scanning Machines and prepare the models for the Unity Engine
  • Placing the Models precisely, according to a Layout Plan
